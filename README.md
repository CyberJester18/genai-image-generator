# GenAI Image Generator using Stable Diffusion

This is a simple Generative AI project that creates images from text prompts using Stable Diffusion.

## How it works
- User writes a text prompt
- Stable Diffusion generates an image
- The model runs on GPU using Google Colab

## Technologies Used
- Python
- Stable Diffusion
- Hugging Face Diffusers
- Google Colab (GPU)

## Example Prompt
"A peaceful mountain landscape at sunset, soft warm light, calm atmosphere"
